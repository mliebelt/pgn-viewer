Parameter boardSize

The following boards (and moves) set the parameter boardSize:
  • Implicit by using the size of the div
  • Implicit by using the parameter size.
  • Explicit to set the board size (there is no default)
  • Explicit to set the board size big (just to show the difference)
  • Explicit to set the board size smaller (for pgnPrint usage).
  • Explicit to set the board size smaller (for the viewer or editor).
