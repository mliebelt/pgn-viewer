Parameter headers

The parameter headers allows to suppress the headers. It is only relevant for the modes view, print and edit, not used in mode board:
  • If not set, show the headers.
  • Show the headers for print as well.
  • Show the headers in edit mode.
  • Shows that headers may be suppressed.

Normal pgnView, no special parameters
Normal pgnPrint, orientation black, headers are visible
Normal pgnEdit, headers are visible
Normal pgnView, headers: false
Normal pgnBoard, headers are per default not shown
