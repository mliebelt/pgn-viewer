Some games to edit

The following games are examples to show how to edit a game. Working at the moment is:
  • Moves are allowed
  • Moves are checked if possible
  • Comments are displayed and can be changed (and stored back in the internal representation):
    • Move: Starting comment before the move number
    • Before: Comment just before the move
    • After: Comment just after the move
  • Moves are displayed automatically (in the correct notation)
  • Variations are displayed inline, you can jump forth and back at will
  • the PGN button (bottom row right) prints out the current PGN notation, that could be copied back into the source
