Set startPlay in configuration

The following is shown:
  1. default (startPlay not set == starts with start position)
  2. set startPlay == 2
  3. Nf3
  4. d4 (found in variation)
  5. set startPlay == 100 (not found)
  6. a3 (could be played, but not in notation)
Useful in all mode view, edit, but not for board, print. Shows an error for non-existing move.
