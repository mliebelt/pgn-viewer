Ticket 171 at GitHub. Use the developer tools of the browser to do the following:

  1. Set a breakpoint after the pgnView (or pgnEdit).
  2. Type into the console: v.base.manualMove(SAN) where SAN is the SAN string for a move that is possible in that position.
  3. Then continue, but leave the console open.
  4. See how the move is made on the board.
  5. Type some more moves the same way.
