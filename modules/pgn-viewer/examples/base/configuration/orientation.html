parameter orientation: "white" (default) or "black"

The parameter "orientation" denotes the orientation of the board. The default value "white" defines that the white pieces are at the bottom, the value "black" defines that the black pieces are at the bottom.
  1. The first setup is with explicit orientation "white".
  2. The next is with the implicit orientation "white" (as default value for the parameter)
  3. The last one is with the explicit orientation "black" to hide the notation.
