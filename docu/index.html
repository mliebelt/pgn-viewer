About

PgnViewerJS was born as the third or fourth attempt to get a nice PGN viewer for our chess club web site. PGN stands for Portable Game Notation which was born in the late 80s, early 90s of the last century. I provided some of the translations there ...

I first started using pgn4web and others as part of our Wordpress Installation, but I had difficulties here and there. At the end, I wanted to have a viewer that was at least as nice as the one from chess.com

Implementation

The implementation contains the following parts:

  • Chess.js: Small JavaScript library that provides the base functionality for playing chess (but without a board).
  • Chessboardjs: Builds the missing building blocks for displaying a chess board, with some functions: Moving pieces, some animations, ...
  • jQuery
  • PEG.js: Parser Generator for JavaScript to read pgn notation.
  • PgnViewerJS: builds the wrapper, makes the move output, allows some different interfaces: pgnBoard, pgnPrint, pgnView, pgnEdit

How To & Support

To start with PgnViewerJS, you have to do the following steps:

  • Download the current sources or the distro from below.
  • Fill the head of your HTML file with the relevant definitions, examples are included in the distro in the directory examples.
  • Define in the body 2 parts:
    • A div section with a unique ID.
    • A script section that defines which API of the viewer should be used. See the examples section how to do it.
  • Fill the configuration of the PgnViewerJS with the relevant aspects: starting position, moves, theme, piece style, size, ... You may use the configuration builder to try out which options are possible.
